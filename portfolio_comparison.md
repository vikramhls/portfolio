# 🎨 Portfolio Design Comparison: Galekto vs Jackie Zhang

## Side-by-Side Overview

| Aspect | **Galekto.com** | **JackieZhang.co.za** |
|--------|-----------------|----------------------|
| **Aesthetic** | 🖥️ Retro-futuristic, Terminal/Cyberpunk | ☀️ Warm, Minimal, Personal |
| **Color Scheme** | Pure black + white + terminal green | Neutral tones, warm accents |
| **Typography** | Monospace + geometric sans-serif | Clean modern sans-serif |

| **"Wow" Factor** | 🔥🔥🔥🔥🔥 Extremely high | 🔥🔥🔥 Moderate |
| **Complexity to Build** | Very High | Moderate |

## 🏆 Verdict: **Use a MIXTURE — Jackie Zhang as Base + Galekto's Best Elements**

> [!IMPORTANT]
> For a **CSE 4th year student targeting placement/jobs**, recruiter-readability is MORE important than raw visual wow. Jackie Zhang's layout wins here. But Galekto has elements that can elevate your portfolio.

### What to Take from Jackie Zhang (Base) ✅
| Element | Why |
|---------|-----|
| **Single-page scroll layout** | Recruiters won't click through 4 pages — they want everything in one scroll |
| **Clean hero with dynamic text** | e.g., *"I build with React / Python / Blockchain"* — immediately tells who you are |
| **Personal "about me" marquee** | Makes you memorable — add fun facts about yourself |
| **Spacious typography** | Easy to read, professional, modern feel |
| **Direct CTA at the bottom** | Simple "Let's connect" with email/LinkedIn links |

### What to Take from Galekto (Enhancements) 🔥
| Element | Why |
|---------|-----|
| **Dark theme color scheme** | Black background looks premium and is trendy for dev portfolios |
| **Row-based project list** | Much cleaner than cards for showing 3-4 projects with tech stack + year |
| **"Journey" timeline section** | Perfect for showing your education, internships, hackathons |
| **Monospace accents** | Use monospace font for code-related labels — screams "developer" |
| **Subtle terminal-style micro-animation** | A **small** typing effect on the hero (not a full boot sequence) adds personality without blocking content |

### What to SKIP ❌
| Element | Why |
|---------|-----|
| Galekto's boot sequence loader | Blocks content — recruiters will leave in 3 seconds |
| Galekto's multi-page navigation | Too complex for a student portfolio |
| Galekto's artworks section | You're not a designer — focus on projects |
| Galekto's coordinates/Easter eggs | Fun but unnecessary complexity |
